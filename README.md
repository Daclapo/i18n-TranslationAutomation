# i18n-TranslationAutomation

A specialized Python CLI tool to automate i18n translation workflows for multilingual web app development, making it easy to localize your website into many languages while only writing in your source language.

Currently, there are three operation modes available after setup: an inline/single command mode, an interactive prompt mode, and a bulk CSV/JSON import mode. Using the translation API of your choice (DeepL or Azure), it translates your source text into as many target languages as needed, automatically building a ready-to-use messages.json file, complete with automatic backups and changelogs.
